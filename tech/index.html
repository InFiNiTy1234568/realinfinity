Mój tech stack

Poznaj mój skillset i narzędzia przeze mnie używane

NodeJS

Framework przeze mnie używany do wykonywania aplikacji backendowych ( botów Discord, API, stron internetowych )

HTML5

Język znaczników ( nie, nie programowania ), którego używam do tworzenia prostych stron internetowych

CSS3

Język stylowania, którego głównym zadaniem jest modyfikacja struktury HTML, nadając jej wygląd

JavaScript / TypeScript

Język programowania głównie przeze mnie używany we wszystkich aktualnych projektach

Python

Język, którego używam głównie do prostych aplikacji, automatyzacji gier czy tworzenia interfejsów graficznych

React

Biblioteka JavaScript umożliwiająca wygodne tworzenie szybkich i skalowalnych aplikacji

Visual Studio Code

Według mnie najlepszy edytor kodu, nie tylko dla laików programowania czy vibe coderów, lecz również programistów ceniących sobie prostotę i wygodę użytkowania oraz wydajność

MongoDB

System bazodanowy bazujący na nierelacyjnym zapisie danych. Struktura zapisywanych informacji jest przechowywana w dokumentach bardzo przypominających obiekt JSON

GitHub

Usługa umożliwiająca programistom przechowywanie, zarządzanie i współpracę nad kodem źródłowym projektów programistycznych.
